## 🚀 ماذا تفعل الآن؟

### الخطوة 1: رفع سياسة الخصوصية (15 دقيقة)

**أ) عدّل الملف:**
```
1. افتح: privacy-policy/index.html
2. عدّل معلومات الاتصال
3. احفظ
```

**ب) ارفع على GitHub:**
```bash
cd privacy-policy
git init
git add .
git commit -m "Add privacy policy"
git branch -M main
git remote add origin https://github.com/اسمك/sanadi-privacy.git
git push -u origin main
```

**ج) فعّل GitHub Pages:**
```
1. اذهب إلى GitHub > Repository
2. Settings > Pages
3. اختر: Branch: main, Folder: / (root)
4. احفظ
5. الرابط: https://اسمك.github.io/sanadi-privacy/privacy-policy/
```

### الخطوة 2: تحديث Google Play Console (10 دقائق)

**أ) رابط سياسة الخصوصية:**
```
1. اذهب إلى: App content > Privacy policy
2. أضف الرابط من الخطوة 1
3. احفظ
```

**ب) Data Safety Section:**
```
اذهب إلى: App content > Data safety

أضف البيانات التالية:
✅ Location (Precise & Approximate)
✅ Health data (Medication records فقط - لا تضف Heart Rate)
✅ Photos
✅ Personal info (Name, Email, Phone)
✅ App activity (Analytics)
✅ Device IDs

لكل نوع:
- مشفر؟ نعم
- يمكن حذفه؟ نعم
- السبب؟ App functionality
```

**ج) Background Location Declaration:**
```
اذهب إلى: App content > Sensitive permissions > Background location

اكتب:
"نستخدم الموقع في الخلفية لميزة الطوارئ. عند الضغط على زر الطوارئ،
يُرسل موقع المستخدم فوراً لجهات الاتصال الطارئة (الأقارب).
هذه ميزة أساسية لسلامة كبار السن."
```

### الخطوة 3: تحديث الكود (20 دقيقة)

**أ) استخدام نظام الأذونات:**

في أي مكان تطلب فيه أذونات، استبدل:

```dart
// ❌ القديم
await Permission.location.request();

// ✅ الجديد
import 'package:sanadi/features/permissions/utils/permission_helper.dart';

await PermissionHelper.requestLocationPermission(context);
```

**ب) أمثلة الاستخدام:**

```dart
// 1. ميزة الطوارئ (Emergency)
final locationGranted = await PermissionHelper.requestLocationPermission(context);
if (locationGranted) {
  final bgGranted = await PermissionHelper.requestBackgroundLocationPermission(context);
}

// 2. تذكير الأدوية (Medications)
final granted = await PermissionHelper.requestNotificationPermission(context);

// 3. الصورة الشخصية (Profile)
final granted = await PermissionHelper.requestPhotosPermission(context);
```

**ج) إضافة رابط سياسة الخصوصية في التطبيق:**

في شاشة الإعدادات:

```dart
import 'package:url_launcher/url_launcher.dart';

const privacyUrl = 'https://اسمك.github.io/sanadi-privacy/privacy-policy/';

ListTile(
  leading: Icon(Icons.privacy_tip),
  title: Text('سياسة الخصوصية'),
  onTap: () async {
    final uri = Uri.parse(privacyUrl);
    if (await canLaunchUrl(uri)) {
      await launchUrl(uri, mode: LaunchMode.externalApplication);
    }
  },
),
```

## ⚠️ ملاحظات مهمة جداً

### 1. قياس ضربات القلب
- ❌ **لم تستخدمه** في التطبيق
- ❌ **لا تذكره** في سياسة الخصوصية
- ❌ **لا تذكره** في Data Safety Section
- ✅ الكود موجود لكن غير مستخدم (للمستقبل)

### 2. إذن الكاميرا
- ❌ **تم إزالته** من AndroidManifest
- ✅ الكود موجود في PermissionHelper (للمستقبل)
- ⚠️ إذا أردت استخدامه لاحقاً، أضف الإذن مرة أخرى

### 3. البيانات الصحية في Data Safety
**اذكر فقط:**
- ✅ Medication records (سجل الأدوية)
- ✅ Medical appointments (المواعيد الطبية)

**لا تذكر:**
- ❌ Heart Rate
- ❌ Vital signs
- ❌ أي شيء متعلق بالكاميرا
